# React-Othello

## 環境
- React.js v16.13.1
- Node v10.x.x

## 導入
```
yarn
```

※yarnコマンドが無ければインストールしてください
```
npm install -g yarn
```

## Tasks
```
yarn start
```
webpack-dev-serverを立ち上げ、`js`,`sass`ファイルの監視をし、変更箇所を自動で反映させます。

```
yarn build
```
production用にminyfyされた`js`, `css`, `html`ファイルを生成します。
